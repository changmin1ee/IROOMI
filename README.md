[IROOMI]

AI chatbot web service built as a team project for TABA 2nd cohort.

[Overview]

Developed a chatbot web service using the ChatGPT API. The backend was built with Spring Boot and deployed on AWS.

[Tech Stack]


Backend: Java, Spring Boot
Frontend: HTML, CSS, JavaScript
AI: ChatGPT API (OpenAI)
Infra: AWS
Build: Maven


[Features]


Real-time AI conversation using ChatGPT API
Web-based chat interface
RESTful API server
Cloud deployment via AWS

[개요]

ChatGPT API를 연동하여 사용자와 자연스러운 대화가 가능한 챗봇 웹 서비스를 개발하였으며, Spring Boot 백엔드 서버를 구축하고 AWS를 통해 배포까지 진행하였습니다.

[기술 스택]


백엔드: Java, Spring Boot
프론트엔드: HTML, CSS, JavaScript
AI: ChatGPT API (OpenAI)
인프라: AWS
빌드 도구: Maven


[주요 기능]


ChatGPT API를 활용한 실시간 AI 대화
웹 기반 챗봇 인터페이스
Spring Boot 기반 RESTful API 서버
AWS를 통한 클라우드 배포
